## Requirements for development

Install ESP8266 Arduino 
- https://github.com/esp8266/Arduino#installing-with-boards-manager
- Install board in Board Manager: esp8266

Install the following libraries in Arduino IDE:
- ArduinoOTA
- ArduinoJson
- Blynk
- PubSubClient
